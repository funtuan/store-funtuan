## 三、技術規格

### 前端框架
**選用：Astro**

**理由**:
1. 專為內容型網站設計，生成靜態 HTML
2. 與 Cloudflare Pages 完美整合
3. 效能優異，預設零 JavaScript
4. 支援多種 UI 框架（可選擇性加入 React/Vue 元件）
5. 開發體驗良好，適合快速迭代

**替代方案**: Next.js (SSG 模式) 或 React + Vite

### 部署平台
- **Cloudflare Pages**
- 自動 CI/CD（連接 Git repository）
- 免費 SSL 憑證
- 全球 CDN 加速

### 技術要求
- 響應式設計（RWD）- 支援手機、平板、桌面
- 現代化瀏覽器支援
- 快速載入速度
- 無障礙設計（Accessibility）

### 不需要的功能
- ❌ SEO 優化（現階段）
- ❌ 購物車系統
- ❌ 會員系統
- ❌ 訂單管理系統
- ❌ 金流串接（現階段）

## 四、設計風格指引

### 色彩使用
- **主色調** (#D4A373)：標題、重點區塊背景、主要按鈕
- **輔助色** (#A3CEF1)：裝飾元素、次要按鈕、卡片邊框
- **白色** (#FFFFFF)：背景、卡片底色

### 視覺風格
- 簡約現代
- 溫暖友善
- 清晰易讀
- 適度的圓角設計（8-12px）
- 柔和的陰影效果

### 字型建議
- **中文**: Noto Sans TC / 思源黑體
- **英文**: Inter / Poppins
- **標題**: 粗體（Bold 700）
- **內文**: 一般（Regular 400）

### 間距與佈局
- Section 間距: 80-120px
- 卡片間距: 24-32px
- 內容最大寬度: 1200px
- 內容左右 padding: 24px（手機）/ 48px（桌面）
